# Melodic HTML VS Code Extension

HTML/CSS IntelliSense for Melodic `html` and `css` tagged templates.

## Features

- Syntax highlighting for `html`/`css` tagged templates in JS/TS/TSX.
- HTML attribute and tag IntelliSense inside `html` templates.
- CSS property IntelliSense inside `css` templates.
- Supports Melodic binding prefixes: `@event`, `.prop`, `:action`.

## Development

```bash
npm --workspace @melodicdev/melodic-html run build
```

To test locally, use VS Code's "Extension Development Host" and point it at `packages/melodic-html`.
